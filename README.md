# Price Scraper

A Python web scraper that extracts book titles and prices from [books.toscrape.com](https://books.toscrape.com/) and saves them to a CSV file.

## Libraries Used
- `requests` — fetches the webpage
- `beautifulsoup4` — parses the HTML
- `csv` — saves the data to a CSV file

## How to Run

1. Install the required libraries:
   pip install requests beautifulsoup4

2. Run the scraper:
   python "Price Scraping Project.py"

3. Output will be saved to `books.csv`
